**Задача**: Решить $a \cdot \ddot{x} + b \cdot \dot{x} + c \cdot x = d$  
**Решение**:  
Была написана функция  
```python
def equation_dynamics(x):
    """
    Equation dynamics: d²θ/dt² = -1/a * (b * dθ/dt + c * θ - d)
    State vector x = [θ, dθ/dt]
    """
    a = -1.68
    b = 0.53
    c = 6.69
    d = 9.62
    
    theta = x[0]
    theta_dot = x[1]
    
    theta_ddot = -1/a * (b * theta_dot + c * theta - d)
    
    return np.array([theta_dot, theta_ddot])
```
